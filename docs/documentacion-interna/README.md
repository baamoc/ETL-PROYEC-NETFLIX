# DW Netflix - Documentacion interna

Este directorio concentra la documentacion interna del proyecto.

## Orden recomendado

1. `indice.md`
2. `guias/como-usar.md`
3. `seguimiento/resumen-avance.md`
4. `reglas/decisiones-tecnicas.md`
5. `guias/guia-colaboracion.md`
6. `reglas/reglas-trabajo-equipo.md`
7. `referencias/consigna.md`
8. `referencias/informe-oficial.md`
9. `referencias/documentacion-interna-completa.md`

## Mapa rapido

- `indice.md`: mapa principal de navegacion.
- `guias/`: uso de la documentacion interna y guia breve para el equipo.
- `reglas/`: decisiones tecnicas y reglas de trabajo.
- `seguimiento/`: resumen operativo y registros por fecha.
- `referencias/`: consigna, informe oficial y documentacion extensa.

## Convenciones

- Titulos en formato `#` y `##`.
- Usar listas cortas y directas.
- Evitar duplicar informacion: usar como referencia principal `referencias/documentacion-interna-completa.md`.
- Registros: usar formato `YYYY-MM-DD-titulo.md`.
