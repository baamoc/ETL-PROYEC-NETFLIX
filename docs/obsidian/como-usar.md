# Como usar este vault

Este vault esta pensado para que cualquier companero entienda el estado del proyecto en 5 minutos.

## Si eres nuevo

1. Lee `indice.md`.
2. Abre `estado-operativo.md` para entender el estado real.
3. Revisa `resumen-avance.md` para saber que sigue.
4. Consulta `decisiones-tecnicas.md` para no romper reglas.
5. Usa `guia-colaboracion.md` y `reglas-trabajo-equipo.md` para trabajar en equipo.

## Si vas a trabajar en ETL

1. Revisa `resumen-avance.md`.
2. Confirma reglas en `decisiones-tecnicas.md`.
3. Documenta evidencias en `docs/evidencias/`.
4. Registra avances en la bitacora de la fecha.

## Si vas a documentar

1. Actualiza `estado-operativo.md` y `resumen-avance.md`.
2. Si es un cambio grande, crea o edita una bitacora.
