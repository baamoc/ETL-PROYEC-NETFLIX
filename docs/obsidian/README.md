# DW Netflix - Vault Obsidian

Este directorio esta organizado como un vault de Obsidian para documentar el proyecto.

## Orden recomendado

1. `indice.md`
2. `como-usar.md`
3. `estado-operativo.md`
4. `resumen-avance.md`
5. `decisiones-tecnicas.md`
6. `guia-colaboracion.md`
7. `reglas-trabajo-equipo.md`
8. `consigna.md`
9. `informe-oficial.md`

## Mapa rapido

- `indice.md`: mapa principal de navegacion.
- `como-usar.md`: como se usa el vault.
- `estado-operativo.md`: estado real y operativo del proyecto.
- `resumen-avance.md`: resumen corto y accionable del avance.
- `decisiones-tecnicas.md`: reglas y decisiones tecnicas.
- `guia-colaboracion.md`: guia breve para el equipo.
- `reglas-trabajo-equipo.md`: flujo de trabajo y que se documenta.
- `consigna.md`: consigna oficial del curso.
- `informe-oficial.md`: apunta al informe oficial.
- `bitacoras/`: cierres y registros por fecha.
- `archivo/`: documentos largos o historicos.

## Convenciones

- Titulos en formato `#` y `##`.
- Usar listas cortas y directas.
- Evitar duplicar informacion: enlazar con `[[archivo]]`.
- Bitacoras: usar formato `YYYY-MM-DD-titulo.md`.
